# [Git History](https://github.com/pomber/git-history) #

快速浏览文件修改记录的 Git History

如上图所示，该工具的功能和使用方法简洁明了：
1、将任何文件的 URL 中的“github.com”替换成“github.githistory.xyz”即可
2、没有第二步

只要一步操作即可以动画的方式快速查看该文件的修改历史记录
